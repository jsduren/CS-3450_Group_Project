About

MECH-Armies is a real time strategy game, with the object to destroy the enemy base.

How to play

Move the player jet moving the controls listed below. Shoot enemy attackers with the player jet or the MECH. Order different cargo units, and set up their functions via control menu. Transport units with the jet. You need to strategize a way to take over intermediate bases to advance across the map, and destory the enemy base without running out of energy or having your home base destroyed by the enemy.
Unit Programs
  • Stand Ground: The unit will not move from this point and shoot at any opposing unit that comes close to it
  • Guard: The unit will move to attack anything that comes near it
  • Attack Main: The unit will find the other players main base and move to attack it
  • Attack Nearest Object: The unit will move to find the nearest base and attack it
Take over a base
To take over a base a player must deploy 4 infantry that make it to the center of the base. The main base of the opposing player cannot be take over, it may only be destroyed. The lights on the base will indicate how many of infantry units are in the base.

Controls

Movement
up arrow / 'w' key advance up
left arrow / 'a' key advance left
down arrow / 's' key advance down
right arrow / 'd' key advance right

Player Options
space bar shoot
t Transform to Mech / Ship (NOT ACTIVE)
c Drop Cargo Unit
v Pick up Cargo Unit (NOT ACTIVE)

Order Menu
left ctrl show menu / hide menu
left arrow menu left
right arrow menu right
up arrow menu up
down arrow menu down
space create unit
